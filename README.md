# DE-Scraping-API
Проект по Web Scraping и API-парсингу для двух выбранных сайтов с тематикой игр (на различных платформах): rawg.io (API) c оценками игр от пользователей и gamalytic.com (Web Scraping + Selenium) с выручкой игр.

1. Произведена первичная обработка (очистка) данных: реализовали алгоритм сопоставления одинаковых игр с разными названиями из двух датасетов (для корректного джоина) и убрали строки с нулевым рейтингом (не представляют ценности для анализа от их выручки).
